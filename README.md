# ترفة ستور - شهاب مجلي 🏪

**نظام إدارة المبيعات والمخزون الاحترافي**

نظام متكامل لإدارة المتاجر والمبيعات والمخزون بواجهة عربية سهلة واحترافية.

## ✨ المميزات

- 🛒 نظام نقاط بيع (POS) سريع وسهل
- 📦 إدارة المخزون مع تنبيهات النواقص
- 👥 إدارة العملاء والموردين مع كشوفات حساب
- 📊 تقارير متقدمة قابلة للطباعة والتصدير
- 💰 إدارة الصندوق والخزينة
- 🔐 نظام صلاحيات المستخدمين
- 🖨 طباعة الفواتير والتقارير

## 💻 متطلبات التشغيل

- Windows 7/8/10/11
- .NET Framework 4.7.2
- SQL Server / SQL Express
